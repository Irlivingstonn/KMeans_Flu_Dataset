Assignment #1 Flu Data Clustering

Description: This project implements kmeans and fuzzy cmeans clustering on the flu data

The project tests clustering using different combinations of features and different values of k (number of clusters). Additonally evaluation is done with 3D scatter plots and using the Dunn Index.
